# Hero Survivor

Jogo web 2D de sobrevivência feito com **Phaser**, **TypeScript** e **Firebase**.

## 🎮 Jogar online

Acesse o jogo aqui:

https://heroes-survivor.vercel.app

## 🕹️ Sobre o jogo

Em **Hero Survivor**, você escolhe um personagem, enfrenta monstros, coleta XP, evolui armas e tenta sobreviver o máximo possível.

O jogo também possui modo online, onde é possível criar uma sala, compartilhar o código com outros jogadores e disputar uma batalha em arena.

## ✨ Funcionalidades

- Modo solo.
- Modo online com criação de sala.
- Sistema de personagens.
- Habilidades especiais.
- Armas evolutivas.
- Monstros e coleta de XP.
- Arena online com zona fechando.
- Deploy automático pela Vercel.

## 🧙 Personagens

- **Knight** — resistente, usa escudo e ataques corpo a corpo.
- **Ranger** — rápido, usa dash e ataques à distância.
- **Mage** — mágico, usa explosões e poderes em área.

## 🛠️ Tecnologias

- Phaser
- TypeScript
- Vite
- Firebase
- Firestore
- Vercel

## 🚀 Rodar localmente

Clone o projeto:

```bash
git clone https://github.com/Zaion07/Hero_Suvivor.git
```

Entre na pasta:

```bash
cd Hero_Suvivor
```

Instale as dependências:

```bash
npm install
```

Rode o projeto:

```bash
npm run dev
```

## 📦 Build

```bash
npm run build
```
